# @ims/web-finance

**Type**: Web App (Next.js 15)  
**Port**: 3013  
**Standard/Domain**: Finance  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Finance management interface with general ledger, invoicing, budgets, purchase orders, tax, banking, and financial reports.

## Running standalone

```bash
pnpm --filter @ims/web-finance dev
```
