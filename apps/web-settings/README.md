# @ims/web-settings

**Type**: Web App (Next.js 15)  
**Port**: 3004  
**Standard/Domain**: System Settings  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

System administration interface for user management, roles, API keys, feature flags, SSO/SAML, SCIM, webhooks, and platform configuration.

## Running standalone

```bash
pnpm --filter @ims/web-settings dev
```
