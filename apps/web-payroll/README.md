# @ims/web-payroll

**Type**: Web App (Next.js 15)  
**Port**: 3007  
**Standard/Domain**: Payroll  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Payroll management interface with pay runs, salary structures, tax calculations, benefits, expenses, and multi-jurisdiction support.

## Running standalone

```bash
pnpm --filter @ims/web-payroll dev
```
