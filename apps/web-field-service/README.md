# @ims/web-field-service

**Type**: Web App (Next.js 15)  
**Port**: 3023  
**Standard/Domain**: Field Service  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Field service interface for job dispatching, technician scheduling, route planning, contracts, and time tracking.

## Running standalone

```bash
pnpm --filter @ims/web-field-service dev
```
