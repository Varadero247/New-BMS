# @ims/web-health-safety

**Type**: Web App (Next.js 15)  
**Port**: 3001  
**Standard/Domain**: ISO 45001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Health and safety management interface for ISO 45001 — risk assessments, incident reporting, CAPA tracking, legal compliance, training, and safety metrics.

## Running standalone

```bash
pnpm --filter @ims/web-health-safety dev
```
