# @ims/web-food-safety

**Type**: Web App (Next.js 14)  
**Port**: 3020  
**Standard/Domain**: ISO 22000  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Food safety management interface for ISO 22000/HACCP — CCPs, hazards, allergens, traceability, recalls, and sanitation.

## Running standalone

```bash
pnpm --filter @ims/web-food-safety dev
```
