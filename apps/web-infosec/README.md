# @ims/web-infosec

**Type**: Web App (Next.js 15)  
**Port**: 3015  
**Standard/Domain**: ISO 27001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Information security interface for ISO 27001 — risk register, controls, SOA, incidents, privacy/GDPR, and vulnerability scanning.

## Running standalone

```bash
pnpm --filter @ims/web-infosec dev
```
