# @ims/web-supplier-portal

**Type**: Web App (Next.js 15)  
**Port**: 3019  
**Standard/Domain**: Supplier Portal  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

External supplier self-service portal for documents, NCRs, purchase orders, scorecards, and tickets.

## Running standalone

```bash
pnpm --filter @ims/web-supplier-portal dev
```

- `/scorecard-dashboard`
