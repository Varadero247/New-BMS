# @ims/web-environment

**Type**: Web App (Next.js 14)  
**Port**: 3002  
**Standard/Domain**: ISO 14001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Environmental management interface for ISO 14001 — environmental aspects, events, CAPA, legal compliance, lifecycle analysis, and objectives.

## Running standalone

```bash
pnpm --filter @ims/web-environment dev
```
