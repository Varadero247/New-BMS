# @ims/web-admin

**Type**: Web App (Next.js 14)  
**Port**: 3027  
**Standard/Domain**: Admin Dashboard  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Internal admin dashboard for growth metrics, prospect research, LinkedIn outreach tracking, and monthly business reviews.

## Running standalone

