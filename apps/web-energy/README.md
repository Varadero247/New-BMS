# @ims/web-energy

**Type**: Web App (Next.js 15)  
**Port**: 3021  
**Standard/Domain**: ISO 50001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Energy management interface for ISO 50001 — meters, baselines, EnPIs, SEUs, bills, readings, and improvement projects.

## Running standalone

```bash
pnpm --filter @ims/web-energy dev
```
