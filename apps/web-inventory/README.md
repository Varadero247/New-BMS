# @ims/web-inventory

**Type**: Web App (Next.js 15)  
**Port**: 3005  
**Standard/Domain**: Inventory  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Inventory management interface for products, warehouses, stock levels, transactions, adjustments, and supplier tracking.

## Running standalone

```bash
pnpm --filter @ims/web-inventory dev
```
