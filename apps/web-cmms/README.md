# @ims/web-cmms

**Type**: Web App (Next.js 15)  
**Port**: 3017  
**Standard/Domain**: CMMS  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

CMMS interface for assets, work orders, preventive maintenance, parts, inspections, downtime tracking, and scheduler.

## Running standalone

```bash
pnpm --filter @ims/web-cmms dev
```
