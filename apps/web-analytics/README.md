# @ims/web-analytics

**Type**: Web App (Next.js 15)  
**Port**: 3022  
**Standard/Domain**: Analytics  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Analytics interface with dashboards, KPIs, board packs, predictions, anomaly detection, NLQ, and report scheduling.

## Running standalone

```bash
pnpm --filter @ims/web-analytics dev
```
