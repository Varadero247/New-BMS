# @ims/web-marketing

**Type**: Web App (Next.js 14)  
**Port**: 3030  
**Standard/Domain**: Marketing Site  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Public marketing site with landing pages for 12 ISO standards, ROI calculator, signup flow, and chatbot widget.

## Running standalone

```bash
pnpm --filter @ims/web-marketing dev
```

## Key Pages

- `/iso/[standard]`
- `/roi-calculator`
- `/signup`

## Dependencies

- @ims/types
- @ims/ui
