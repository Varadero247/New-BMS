# @ims/web-iso37001

**Type**: Web App (Next.js 15)  
**Port**: 3025  
**Standard/Domain**: ISO 37001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Anti-bribery management interface for ISO 37001 — risk assessments, due diligence, gift registers, investigations, and training.

## Running standalone

```bash
pnpm --filter @ims/web-iso37001 dev
```
