# @ims/web-customer-portal

**Type**: Web App (Next.js 14)  
**Port**: 3018  
**Standard/Domain**: Customer Portal  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

External customer self-service portal for complaints, documents, invoices, tickets, and NPS surveys.

## Running standalone

```bash
pnpm --filter @ims/web-customer-portal dev
```
