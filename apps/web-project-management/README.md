# @ims/web-project-management

**Type**: Web App (Next.js 15)  
**Port**: 3009  
**Standard/Domain**: Project Management  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Project management interface with projects, tasks, sprints, milestones, Gantt charts, resources, and timesheets.

## Running standalone

```bash
pnpm --filter @ims/web-project-management dev
```
