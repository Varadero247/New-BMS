# @ims/web-hr

**Type**: Web App (Next.js 15)  
**Port**: 3006  
**Standard/Domain**: Human Resources  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Human resources interface for employees, departments, leave, attendance, recruitment, performance reviews, and training management.

## Running standalone

```bash
pnpm --filter @ims/web-hr dev
```
