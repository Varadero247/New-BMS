# @ims/web-esg

**Type**: Web App (Next.js 15)  
**Port**: 3016  
**Standard/Domain**: ESG / CSRD  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

ESG reporting interface tracking emissions, energy, water, waste, social metrics, governance, and sustainability frameworks.

## Running standalone

```bash
pnpm --filter @ims/web-esg dev
```
