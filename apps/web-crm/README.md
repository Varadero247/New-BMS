# @ims/web-crm

**Type**: Web App (Next.js 14)  
**Port**: 3014  
**Standard/Domain**: CRM  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

CRM interface with contacts, accounts, deals, leads, pipeline view, campaigns, quotes, and partner management.

## Running standalone

```bash
pnpm --filter @ims/web-crm dev
```
