# @ims/web-quality

**Type**: Web App (Next.js 14)  
**Port**: 3003  
**Standard/Domain**: ISO 9001  
**Part of**: [Nexara IMS](../../README.md)

## Purpose

Quality management interface for ISO 9001 — audits, nonconformances, CAPA, calibrations, FMEA, document control, and supplier management.

## Running standalone

```bash
pnpm --filter @ims/web-quality dev
```
