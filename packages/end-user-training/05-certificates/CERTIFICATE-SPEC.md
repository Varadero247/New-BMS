# Certificate Specification — End User Training

---

## Certificate Title

**Nexara Platform Foundation — End User Completion Certificate**

---

## Format

- **Size**: A4 landscape (297mm × 210mm)
- **Delivery**: Digital (PDF/PNG) primary; printed optional
- **Paper** (if printed): 170gsm silk minimum
- **Digital resolution**: 300 DPI for print-ready PDF; 150 DPI for on-screen PNG

## Nexara Brand Elements

### Colours
- **Background accent**: Subtle Nexara Navy gradient on left border or header band (`#1E3A5F`)
- **Certificate title**: Nexara Navy (`#1E3A5F`), 14pt, Montserrat Bold
- **Programme title**: Nexara Gold (`#B8860B`), 18pt, Montserrat SemiBold
- **Participant name**: Dark (`#1A1A2E`), 24pt, Montserrat Bold
- **Body text**: Slate (`#374151`), 11pt, Montserrat Regular

### Typography
Same as Module Owner Training certificates — Montserrat font family throughout.

## Digital Format Specifics

### PDF Certificate
- Certificate PDF generated by the web portal on assessment pass (portal route: `/end-user/assessment`)
- Filename: `nexara-foundation-{YYYYMMDD}-{participant-last-name}.pdf`
- PDF/A-1b format for long-term archival

### QR Code Verification
- Each digital certificate includes a QR code linking to the verification registry
- QR code position: bottom-left, 25mm × 25mm
- Verification URL: `https://verify.nexara.io/cert/{certificate-id}`
- Certificate ID format: `NEU-{YEAR}-{6-DIGIT-RANDOM}` (e.g. `NEU-2026-837491`)

### Bulk Digital Distribution
For large deployments (100+ participants), certificates can be batch-generated via the Nexara Training Registry API. Contact training@nexara.io for batch generation access.

---

## Organisation Name Field

The `Organisation` field on the certificate should contain the employer's organisation name, as configured by the Nexara administrator in the training portal settings.
