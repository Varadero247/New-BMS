# Competency Checklist — End User Training

*For completion by the participant's line manager as on-the-job verification that the participant can perform the key tasks in a live Nexara IMS environment.*

---

## Instructions for Line Managers

This checklist is completed within **30 days** of the participant completing the End User Training Programme. Observe the participant performing each task in the live Nexara environment (not the training sandbox). Tick each item as you observe the participant completing it correctly and independently.

**When complete**: Upload the signed checklist to the participant's training record in Nexara HR → Employee → Training History → [End User Training] → Attach Competency Checklist.
