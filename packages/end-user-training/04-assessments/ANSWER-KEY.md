# ANSWER KEY — End User Summative Assessment

**FACILITATOR ONLY — Do not distribute to participants**

| Q | Answer | Explanation |
|---|--------|-------------|
| 1 | B | Training module (graduation cap icon in sidebar) is where all assigned training lives. Not in H&S, not in profile settings. |
| 2 | C | Incident (harm occurred — sprain, medical treatment) + MODERATE (medical treatment beyond first aid; no hospitalisation). MINOR = first aid only. |
| 3 | C | dateOccurred = when it happened. Entering today's date is a common error that distorts incident trend data. |
| 4 | B | Under Investigation = HSE manager has opened a formal investigation. This is normal and expected for recordable incidents. |
| 5 | C | Amber = due within 30 days. Act now, before it becomes red (overdue). |
| 6 | C | Self-record external completions: Training → course → Record External Completion → details + certificate. HR does not do this automatically. |
| 7 | B | Confined space entry is a standard PTW requirement. Routine maintenance, ground-floor window cleaning, and hand drilling do not meet PTW triggers. |
| 8 | C | Read the permit conditions (approvers may have added conditions) and confirm pre-work checks in the system before starting work. |
| 9 | B | Permits expire at their stated end time. Stop, secure, reapply before expiry. Never continue under an expired permit. |
| 10 | B | Closure confirmation: work complete + area safe + no hazards. The permit is now formally closed. |
| 11 | C | Read the rejection reason, fix the issue (rewrite with specific details), resubmit. Never proceed without an approved permit. |
| 12 | C | Observation (negative) — a hazard condition has been identified but no incident has occurred. Reporting it triggers a corrective action to prevent a slip. |
| 13 | C | Stopping work due to safety concern = positive safe behaviour. The other three are all negative observations (hazard conditions). |
| 14 | B | Anonymous = your name doesn't appear, but the report is still received and actioned. It is not lost. You lose the ability to track status. |
| 15 | C | Red = at least one mandatory item overdue or expired. Not password, login frequency, or contract dates. |
| 16 | C | The fastest and most credible response: show the live Nexara training history directly. More reliable than email or verbal. |
| 17 | C | My Incident Records report = incidents and near misses submitted by you. Observations and permits are in separate reports. |
| 18 | C | Complete a refresher before expiry to maintain green status. Ignoring it will result in red status and an overdue notification to your manager. |
| 19 | B | Training completion in Nexara requires an explicit action (click "Complete Course" or submit a knowledge check). Watching/reading alone does not register. |
| 20 | C | Personal compliance dashboard = your individual status: training, actions, submissions. Not for team management or financial data. |
