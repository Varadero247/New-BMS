# Virtual Facilitation Guide — End User Training

---

## Zoom/Teams Setup

### Before the Session (30 minutes prior)

1. **Start the meeting early** (30 minutes before) — verify recording, polling, and screen share
2. **Open sandbox in a separate browser** — tab ready for each module's starting point
3. **Pre-create polls** (see poll questions below)
4. **Set up breakout rooms** for Module 2 exercise (groups of 4–5)
5. **Test audio** with a co-facilitator
6. **Admit attendees from waiting room** at 08:55 — greet each by name as they join

### Co-Facilitator Role (Recommended for 15+ participants)

The co-facilitator monitors the chat and helps participants who are struggling:
- Answer technical questions in chat while the main facilitator delivers content
- Spot participants who are not following along (no camera, no chat activity)
- Direct message participants with issues
- Manage breakout rooms during exercises

## Screen Share Structure

### Best Practice: Two-Monitor Setup

- Monitor 1: Nexara sandbox (shared with participants)
- Monitor 2: Facilitation notes and chat (not shared)

If single-monitor: use "Share Window" (not "Share Desktop") to avoid showing your facilitator notes. Close all other browser tabs before sharing.

### What to Show on Screen (by Module)

| Module | What to share |
|--------|--------------|
| 1: Navigation | Nexara dashboard — fresh login, no prior navigation |
| 2: Incidents | H&S → Record Incident — blank form |
| 3: Training | Training dashboard — showing assigned items |
| 4: PTW | H&S → Permits → New Permit Request |
| 5: Observations | H&S → Observations → New Observation |
| 6: Reports | Personal dashboard home screen |

## Poll Questions (Use These During the Session)

**Opening poll** (09:05): "Have you used Nexara before?" Options: Yes, regularly / Yes, occasionally / No, this is my first time

**After Module 2** (10:15): "How confident do you feel about recording an incident in Nexara?" Options: Very confident / Fairly confident / Need more practice / Not confident

**After Module 4** (11:35): "Do you currently work with any activities that require a Permit to Work?" Options: Yes, regularly / Yes, occasionally / No, not in my current role

**Closing poll** (12:40): "How useful was today's training?" Options: Very useful — I'll use this immediately / Useful — good to know / Somewhat useful / Not useful for my role

## Guided Exercise Instructions

### Module 2 Exercise Script

"I'd like you all to complete this exercise on your own screen now. Here's the scenario: [Read scenario from VIRTUAL-SESSION-SCHEDULE.md]. Open the Health & Safety module, click Record Incident, and fill in the form using the scenario details. You have 15 minutes. I'll drop into breakout rooms to check in. If you get stuck on any field, put 'HELP: [field name]' in the chat and [co-facilitator] will assist you."

[Set up breakout rooms of 4–5 and send participants in]

[After 12 minutes]: "Two minutes left — please make sure you've hit Submit before I bring you back to the main room."

[Bring back to main room]: "Let's compare results. [Screen share the completed record] — who got this reference number format: HS-INC-[year]-[number]? Thumbs up in the chat."

---

## Chat Rules to Establish at Start

"A few ground rules for chat: please feel free to use it for questions at any time — I or [co-facilitator] will answer them. If you want to ask something that the whole group would benefit from, type QUESTION: before your message. Reactions and emojis are encouraged — they help me know you're following along."

## Handling Technical Issues

| Issue | Response |
|-------|---------|
| Participant can't access sandbox | Co-facilitator direct messages them; if unresolved in 5 min, they observe your screen share for this exercise and try again in the next one |
| Main facilitator's internet drops | Co-facilitator takes over the session; facilitator rejoins within 2 min |
| Portal assessment unavailable at 12:15 | Direct participants to the email backup assessment: send the assessment PDF via chat; they complete and return by email within 2 hours |
| Video call platform crashes | Send fallback meeting link (Teams if Zoom, Zoom if Teams) to all participants via email before the session starts |
