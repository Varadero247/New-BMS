## Quick Start for Facilitators

This package contains all editable source materials for the Nexara IMS Administrator Training Programme. Materials are designed for internal delivery by certified Nexara trainers or licensed client training teams.

### Package Contents

| Directory | Contents |
|-----------|----------|
| `00-programme-overview/` | Programme spec, learning objectives, competency framework, facilitator master notes |
| `01-day1/` | Day 1 schedule, facilitation guide, Modules 1–4 |
| `02-day2/` | Day 2 schedule, facilitation guide, Modules 5–7 |
| `03-participant-handbook/` | Participant workbook, quick-reference card, glossary |
| `04-lab-workbook/` | Lab setup guide + 7 hands-on lab exercises |
| `05-assessments/` | Pre-assessment, formative (Day 1), summative + answer key + marking guide |
| `06-certificates/` | Certificate design spec, co-branding guide, award criteria |
| `07-slide-deck-outlines/` | Slide-by-slide outlines for all 7 modules |
| `08-delivery-guides/` | Virtual and on-site delivery guides, troubleshooting |

### Pre-Delivery Checklist

1. **Read** `00-programme-overview/FACILITATOR-MASTER-NOTES.md` — especially tone and pacing guidance
2. **Review** `00-programme-overview/COMPETENCY-FRAMEWORK.md` — understand observable behaviours
3. **Print** participant handbooks and quick-reference cards (or share digitally)
4. **Set up** lab environment per `04-lab-workbook/LAB-ENVIRONMENT-SETUP.md`
5. **Brief** co-facilitators using `01-day1/DAY1-FACILITATION-GUIDE.md` and `02-day2/DAY2-FACILITATION-GUIDE.md`
6. **Prepare** assessment forms — formative scoring is automatic in the web portal; summative requires manual marking of Part B

### Assessment & Certification Summary

| Assessment | Questions | Scored? | Pass Threshold |
|------------|-----------|---------|----------------|
| Pre-assessment (Day 1 opening) | 20 | No (diagnostic) | N/A |
| Day 1 formative | 15 | Yes | ≥ 70% |
| Summative (Day 2 afternoon) | 40 MCQ + 3 scenarios | Yes | ≥ 75% = Pass; ≥ 90% = Distinction |

### Certificate Grades

- **< 75%**: Fail — retake at next scheduled cohort
- **75–89%**: Nexara Certified Platform Administrator
- **≥ 90%**: Nexara Certified Platform Administrator **with Distinction**

### Editing Guidelines

All content is in Markdown. Edit freely in any text editor or IDE. When updating module content:
- Keep learning objectives aligned with `LEARNING-OBJECTIVES.md`
- Update the `ANSWER-KEY.md` if assessment questions change
- Re-export slide deck outlines to your presentation tool

### Web Portal

The companion web portal (`apps/web-training-portal/`, port 3046) renders this content interactively with online assessments and PDF/PNG certificate generation. See the portal README for deployment instructions.
