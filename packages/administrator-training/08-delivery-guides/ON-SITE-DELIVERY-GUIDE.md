# On-Site Delivery Guide — Nexara IMS Administrator Training

---

## Venue Requirements

| Requirement | Specification |
|-------------|--------------|
| Room capacity | Minimum: cohort size + 2. Maximum: 20 people |
| Room style | Boardroom (≤ 10 pax) or Classroom (≤ 20 pax) |
| Screen/projector | Minimum 2m wide; readable from furthest seat |
| Whiteboard | Full-height whiteboard; 3 colours of marker |
| Internet | Stable WiFi or LAN; minimum 20 Mbps; separate SSID for training if possible |
| Tables | Individual laptop space per participant (600mm wide minimum) |
| Power | Sockets accessible at every seat; extension leads if needed |
| Lighting | No glare on screens; window blinds available |
| Temperature | 19–21°C; adjustable; stuffy rooms destroy afternoon energy |
| Accessibility | Step-free access; accessible bathrooms; font size readable for all |

## Room Layout

### Boardroom Style (≤ 10 participants)
```
[SCREEN]
[FACILITATOR POSITION]

[TABLE: all facing screen]
P P P P P
P P P P P

[Materials table at rear]
```

### Classroom Style (11–16 participants)
```
[SCREEN]
[FACILITATOR POSITION + WHITEBOARD]

[TABLE ROW 1]   P P P P
[TABLE ROW 2]   P P P P
[TABLE ROW 3]   P P P P
[TABLE ROW 4]   P P P P

[Materials table at side]
```

For group exercises (LAB-02), rearrange into clusters of 3–4:
- Boardroom: turn chairs to face each other at ends of table
- Classroom: push 2 tables together per group

## Catering Guidance

| Break | Type | Duration |
|-------|------|---------|
| Morning break | Tea, coffee, water; light snack (biscuits/fruit) | 15 min |
| Lunch | Hot or cold; substantial; dietary requirements pre-confirmed | 45 min |
| Afternoon break | Tea, coffee, water | 15 min |
| Close (Day 2) | Light refreshments; networking | 30 min |

**Dietary requirements**: Confirm with client/venue at least 5 working days before.

**Catering note**: Do not serve heavy lunches — they increase the post-lunch energy dip. Lighter lunches with fruit/protein keep afternoon energy higher.

## Day-of Timeline

### Day 1

| Time | Action |
|------|--------|
| T-60 min | Arrive; room check; AV test; materials laid out |
| T-30 min | Facilitator personal preparation; coffee |
| T-15 min | Participants begin arriving; facilitator greets each one |
| T-0 (08:30) | Welcome; begin programme |
| 10:30 | Arrange morning break refreshments (if not venue-provided) |
| 12:15 | Ensure lunch is ready or being served on time |
| 14:30 | Arrange afternoon refreshments |
| 16:15 | Set up for assessment: remove materials from desks; distribute assessment forms |
| 16:45 | Return assessment materials; begin wrap-up |
| 17:00 | Close; facilitator stays to answer informal questions |

### Day 2

| Time | Action |
|------|--------|
| T-60 min | Arrive; reset room; print results slips |
| T-0 (08:30) | Begin Day 2 |
| 14:00 | Clear desks for assessment; distribute sealed assessment booklets |
| 14:15 | Break seals on assessment booklets |
| 15:45 | Action planning (whiteboard ready) |
