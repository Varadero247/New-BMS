# Module Owner Training — Competency Framework

This framework defines the observable behaviours that demonstrate competence for each module group. Facilitators use these criteria to assess participant progress during lab exercises and to calibrate the assessment question bank.

---

## How to Use This Framework

**During the lab**: Check off observable behaviours as participants demonstrate them in the sandbox.
**During assessment**: Questions are mapped to competency areas to ensure balanced coverage.
**Post-training**: Line managers can use the observable behaviours for on-the-job verification.

Competency levels:
- **Developing**: Can complete the task with facilitator guidance
- **Competent**: Can complete the task independently in the sandbox
- **Proficient**: Can complete the task independently in a live environment and coach others
