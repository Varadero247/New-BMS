# ANSWER KEY — Day C: HR & Payroll

**FACILITATOR ONLY — Do not distribute to participants**

| Q | Answer | Explanation |
|---|--------|-------------|
| 1 | B | HR → Employees → New Employee is the correct path for the master employee record. Settings → Users creates system access, which is a separate step. |
| 2 | C | Best practice: revoke access on the last working day. After hours access is a security risk. The system sends a notification to the admin to complete revocation. |
| 3 | D | UK GDPR / Employment law: employment records + 7 years is standard. Specific records (e.g. accident records under RIDDOR) have separate retention requirements. |
| 4 | B | ValidityPeriod controls certificate validity and refresher scheduling. It does not affect content update schedules or assignment deadlines. |
| 5 | C | Compassionate leave is the correct type for bereavement. Sick leave is for illness. Using sick leave for bereavement misclassifies the absence and affects absence rate KPIs. |
| 6 | B | Red = overdue (deadline passed) or expired (validity period elapsed). Failure, exemption, and missing certificates are shown differently. |
| 7 | C | Lock = all modifications prevented. Only a payroll administrator can unlock the period if a genuine error is found. |
| 8 | B | Always verify overtime claims against approved records before approving pay adjustments. Segregation of duties requires the approver to independently verify, not just trust the entry. |
| 9 | B | 22% variance requires investigation before journal export. Exporting without understanding the variance is a controls failure. |
| 10 | B | ISO 9001 cl. 7.2 requires evidence that people are competent for their roles. The training compliance matrix filtered by role-mandatory courses is the direct evidence. |
| 11 | B | Escalation emails go to HR Manager and Line Manager as configured in notification rules. Line managers must be aware of their team's training compliance. |
| 12 | B | Last day of assignment = access revocation date. Post-assignment access is an information security risk. |
| 13 | C | Journal export = payroll entries for import into the finance system's chart of accounts. Payslips and statutory returns are separate outputs. |
| 14 | B | Create a correction in the following period with a reference. Never modify a Paid period — this would destroy the audit trail. |
| 15 | B | Minimum: name, course, date, result, certificate. ISO 9001 cl. 7.2 requires evidence of competence — date and result are essential to demonstrate that training was completed and effective. |
| 16 | B | Payroll data is restricted by default. HR_VIEWER gives access to attendance, performance, and training — not salary. |
| 17 | B | Return-to-work interviews occur immediately on return, by the line manager (with HR support). Before return would be a pre-return assessment — different record type. |
| 18 | B | Regulatory = legally or regulatorily mandated. The course type affects how it appears in compliance reports and whether its completion is mandatory. |
| 19 | B | Compliance matrix with expiry-date sort is the correct tool. The course filter isolates Moving and Handling; the 60-day expiry filter identifies the at-risk group. |
| 20 | B | BIK is taxable benefit that must be reported to HMRC (P11D). The system generates this data automatically from BIK adjustment records. |
