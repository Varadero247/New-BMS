# Nexara IMS — Module Owner Training Programme

**Version 1.0 | Five One-Day Instructor-Led Programmes | CPD: 7 hours per day**

---

## Overview

The Module Owner Training Programme equips compliance professionals, department heads, and operational managers with the skills to own, operate, and report from their designated Nexara IMS modules. Unlike administrator training (which covers platform configuration), this programme focuses on day-to-day module operation: data entry, workflow management, KPI dashboards, and ISO evidence generation.

## Assessment & Certification

| Criterion | Specification |
|-----------|---------------|
| Format | 20 MCQ per day |
| Time | 30 minutes (timed) |
| Pass threshold | ≥ 75% (15/20) |
| Grade | Pass or Fail — no distinction grade for one-day programmes |
| Retake | Next scheduled cohort |
| Certificate | Issued same day on pass |
| CPD hours | 7 per day |

## Pre-Delivery Checklist

1. **Read** `00-programme-overview/FACILITATOR-MASTER-NOTES.md` for audience and tone guidance
2. **Review** `00-programme-overview/COMPETENCY-FRAMEWORK.md` for observable behaviours
3. **Select** the relevant day programme based on participant roles
4. **Set up** the Nexara IMS sandbox environment with sample data pre-loaded
5. **Print** or share the relevant sections of `06-participant-materials/PARTICIPANT-HANDBOOK.md`
6. **Prepare** lab scenario data per the instructions in each `LAB-DAY-*.md` file

## Editing Guidelines

All content is in Markdown. Edit freely in any text editor or IDE. When updating:

- Keep question banks in sync with corresponding `ANSWER-KEY-DAY-*.md` files
- Preserve the 20-question count per assessment day
- Do not remove CPD hour attribution lines from schedule files
- Update glossary terms when new field names are added in IMS releases
