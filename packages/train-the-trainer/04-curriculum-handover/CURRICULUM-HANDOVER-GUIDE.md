# Curriculum Handover Guide

**Programme**: Nexara IMS Train-the-Trainer (T3)
**Version**: 1.0
**Timing**: End of Day 2, following certificate ceremony

---

## 1. Overview

On successful completion of the Train-the-Trainer programme, each certified internal trainer receives the Nexara curriculum handover package. This is a formal transfer of the source materials needed to independently deliver End User Training and Module Owner Training to internal cohorts.

The handover is not simply a file transfer — it is a formal, documented process with a transfer checklist, version register, and signed licence acknowledgement.

## 3. Handover Process

### Step 1 — Transfer Method

Materials are transferred via one of two methods:

| Method | When Used | Details |
|--------|----------|---------|
| **USB drive** | In-person cohort | Trainer receives a pre-prepared USB drive at end of Day 2; USB is marked with the trainer's name, cohort date, and version number |
| **Secure download link** | Virtual cohort | Nexara L&D emails a time-limited secure download link within 2 hours of Day 2 completion; link expires after 72 hours |

Both methods deliver the same package. The USB or download link contains a compressed archive (`.zip`) of the full package structure.

### Step 2 — Version Register

On receipt of the package, the certified trainer completes the **Version Register** (one page, in the handover confirmation email):

| Field | Value |
|-------|-------|
| Trainer name | |
| Organisation | |
| Package version received | v1.0 |
| End User Training version | v1.0 |
| Module Owner Training version | v1.0 |
| Date of receipt | |
| Transfer method | USB / Download |

The completed register is returned to Nexara by email within **5 business days** of receipt. It becomes part of the trainer's Nexara L&D record.

### Step 3 — Licence Acknowledgement

The handover package includes a **Licence Acknowledgement** document. The certified trainer signs (or e-signs) this document and returns it to Nexara L&D. By signing, the trainer confirms they:

1. Have received the materials listed in this handover guide
2. Understand the licence scope (internal delivery only; named organisation only)
3. Have read and agree to the co-branding rules in `CO-BRANDING-GUIDE.md`
4. Understand the answer key vault access process and will not share answer keys with participants
5. Will notify Nexara within 5 business days of any material version update received
6. Will submit cohort completion data within 5 business days of each cohort

Failure to return a signed Licence Acknowledgement within 10 business days of the handover date is treated as non-acceptance; the trainer account in the Training Portal is suspended pending receipt.

## 5. Post-Handover: What Happens Next

After the handover, the certified trainer should:

| Action | Timing |
|--------|--------|
| Return signed Licence Acknowledgement | Within 10 business days |
| Return completed Version Register | Within 5 business days |
| Set up co-branding in Training Portal | Before first cohort |
| Create first cohort in Training Portal | At least 10 business days before delivery |
| Complete first cohort and submit completion data | Within 5 days of cohort completion |
| Set a calendar reminder for annual renewal | 11 months from certification date |

For questions following the handover: training@nexara.io. Standard response time is 2 business days for non-urgent queries.
